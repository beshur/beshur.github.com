Highlights

Some of the projects I own or relate to on Github:

  • ArduinoBtCar - an Android demo application to control an Arduino car via Bluetooth
  • Toy TraffcLight - a simple Arduino traffic light (blog post)
  • Table Status - a web-app to display a couple widgets for my home (calendar, photos and notes)
  • PlasticPlayer - my take on Brendan Dawes' Plastic Player, using original code for Raspberry Pi and Arduino
  • diUpcomingShows - DI.FM live shows notifications for Discord (and potentially for some other services later)
  • diRandomizer - DI.FM random channel selector
  • Stoplay - a collaborative effort on a Google Chrome extension to control in-browser media playback - can be used to stop/play your music/videos etc.
  • Webhookbot - a bot for Telegram and FB Messenger that creates the webhooks for you and proxies the messages.
  • wp-jouele - a Wordpress plugin for Jouele audio-player
  • learn-it-ukraine - a list of useful resources for learning development (abandoned archived project)
  • getpass - a simple password generator for personal use
  • lifeChart - a small calendar displaying the average weeks count for the current user's life
  • 9gag Pics Only - a Chrome extension to hide all posts except static images on 9gag
